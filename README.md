# Wii U GTX Extractor v3.3
Extracts textures from the GFD ('Gfx2' / .gtx file extension) format used in Wii U games, and saves them as DDS.  
  
Can Also convert DDS files into .gtx files!  

# Requirements:
Python 3.4 or higher.

cx_Freeze. (Optional)
